# Purchase Lists — MVP

MVP веб‑приложение для составления заявок на закупку продуктов по поставщикам.

## Быстрый старт (локально)

```bash
pnpm i   # или npm i / yarn
pnpm dev # или npm run dev
```

Откройте http://localhost:3000

## Деплой на Vercel

1) Создайте новый репозиторий на GitHub и загрузите эти файлы.
2) В Vercel: New Project → Import Git Repository → Framework: Next.js → Deploy.
3) Готово. Env не требуются.

## Формат CSV

Обязательные колонки: `sku, name, category, unit, supplier`  
Необязательные: `pack_size, comment`

Можно русские заголовки: `SKU/sku/id`, `Название`, `Категория`, `ЕдИзм`, `Поставщик`.

## Поток работы

1) `/import` — загрузка CSV, предпросмотр, сохранение в браузер.
2) `/order` — список по категориям, ввод количества.
3) `/review` — группировка по поставщикам, кнопки «Скопировать» и WhatsApp.
